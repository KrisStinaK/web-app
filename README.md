# Новостной блог на Django

Этот проект представляет собой новостной блог, разработанный с использованием Django. Он позволяет пользователям просматривать новости, регистрироваться, публиковать свои статьи и управлять контентом.

## Особенности

- Регистрация и аутентификация пользователей
- Публикация, редактирование и удаление новостей
- Административная панель для управления контентом

## Технологии

- Django 3.x
- Python 3.8+
- SQLite (по умолчанию)
- Bootstrap (для стилизации)

## Установка

1. Клонируйте репозиторий:

```bash
git https://github.com/KrisStinaK/web-app.git
cd project
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Выполните миграции:
```bash
python manage.py migrate
```

## Запуск
Запустите сервер разработки:

```bash
python manage.py runserver
```
Перейдите по адресу http://127.0.0.1:8000/ в браузере.

## Использование

* Зарегистрируйтесь или войдите в систему.
* Добавляйте новости, редактируйте или удаляйте их.
* Используйте административную панель для управления сайтом.
